NMEA-IOT

GPS Tracking & IoT System

Project Description

NMEA-IOT is a marine controller system designed based on the NMEA 2000 standard (CAN protocol) for monitoring and managing vessel information.
The system collects data from onboard sensors, displays real-time information, and communicates with a remote server for data synchronization.

My Responsibilities

  • Implement NMEA 2000 parameters such as fuel level, engine data, and navigation information
  • Render real-time vessel data on the onboard display
  • Develop communication with backend server for data update and monitoring in mobile and web app
  • Research and integrate GNSS positioning using SIM module

Technologies

  • Protocol: NMEA 2000 (CAN)
  • MCU: Renesas
  • GNSS service: AssistNow Online, AssistNow Offline, CLoud-Locate, Cell-Locate
  • Communication: CAN, UART, HTTP, MQTT
  • Language: C
