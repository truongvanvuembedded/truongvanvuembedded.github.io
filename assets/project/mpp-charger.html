MPP Charger System

Developed embedded system to parse NMEA GPS data and transmit to cloud. System runs on ESP32 with FreeRTOS and MQTT.

Main Features

  • NMEA0183 parsing
  • Real-time location tracking
  • MQTT communication
  • Low power mode

Technologies

  • ESP32
  • FreeRTOS
  • C / Python
  • MQTT / TCP-IP
