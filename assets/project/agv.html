AGV

Automated Guided Vehicle

Project Description

The AGV (Automated Guided Vehicle) was developed for factory environments to automate material transportation for workers.
The vehicle follows predefined paths inside the factory to improve efficiency and reduce manual labor.

My Responsibilities

  • Tune and optimize line following algorithm for factory environment
  • Calibrate sensors to improve path detection accuracy
  • Adjust control parameters to ensure stable vehicle movement
  • Debug and validate system performance during real operation

Technologies

  • MCU: STM32 / Arduino
  • Sensors: IR
  • Language: C / C++
