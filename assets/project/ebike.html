Ebike

Vehicle Control Unit

Project Description

Vehicle Control Unit is the main controller of an electric bike system, handling motor inverter control, battery management, vehicle state and user interface.

My Responsibilities

  • Requirement analysis
  • Basic and detailed design
  • Software architecture design
  • Create and implement unit, integration, comprehensive, and system tests
  • Develop low-level layers: I2C, UART, CAN, NVS
  • Develop mid-level layers
  • Develop application layer using UML and state machine
  • Design mechanisms to protect and synchronize data between tasks
  • Design and develop update firmware

Technologies

  • MCU: ESP32
  • RTOS: FreeRTOS
  • Language: C / Python
  • Unit-Test: Ceedling
