MPP Charger System

Maximum Power Point Charger System

Project Description

The MPP Charger System is a bidirectional energy system that supports charging for mobile power supply battery from solar panels or AC/DC sources, and supplies power back to users in both AC and DC modes.
The system is designed with multiple boards working together for power conversion and energy management.

My Responsibilities

  • Develop firmware for the main controller board
  • Implement communication with the battery management board and bidirectional AC/DC converter
  • Design and implement system state management and safety mechanisms
  • Debug and validate system behavior during charging and discharging modes
  • Create multiple simulation and monitoring tools using Python for debugging and evaluation
  • Implement data logging mechanisms to record system data when errors occur in released products
  • Display system information via user interface

Technologies

  • MCU: ESP32, Renesas
  • Communication: CAN, UART, RS485
  • Language: C, Python
  • Domain: Power Electronics, Energy Management
